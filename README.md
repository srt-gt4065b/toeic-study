✅ 주요 변경사항
1️⃣ 5번 틀리면 게임오버

❤️ 하트 5개로 남은 기회 표시
틀릴 때마다 하트가 🖤로 변함
5번 틀리면 "💔 게임 오버" 화면

2️⃣ 틀린 단어 복습

게임오버/미션성공 시 틀린 단어 목록 표시
영어 ↔ 한국어 뜻 함께 보여줌
"다시 틀리지 않도록!" 안내 문구

3️⃣ 모든 문제 풀 때까지 계속

해당 레벨의 모든 단어를 다 풀어야 함
같은 단어 중복 출제 안 함
문제 번호: 1/16, 2/16 ... 형식

4️⃣ 미션 성공 시스템

5개 이내로 틀리고 다 풀면 🏆 미션 성공!
"🚀 다음 레벨 도전!" 버튼 표시
마지막 레벨(990+) 통과 시 "모든 레벨 완료!" 표시

5️⃣ 누적 점수 시스템

각 레벨 통과 점수가 누적 총점에 합산
순위표는 누적 총점 기준으로만 집계
같은 학번은 최고 점수만 표시

6️⃣ 모바일 반응형 처리

화면 크기에 따라 블록 크기 자동 조절
글씨 크기도 비례적으로 축소
scaleFactor = Math.min(1, width / 400) 적용
